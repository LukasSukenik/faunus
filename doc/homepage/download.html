Download

We do not maintain regular releases of Faunus and the code can be accessed only via the subversion repository at Sourceforge with the command:

$ svn checkout https://faunus.svn.sourceforge.net/svnroot/faunus faunus

Alternatively use the code browser to download a tar-ball of any revision.

Currently two versions of Faunus exists: "trunk" and the "jurassic" branch where the latter represents development until 2011.

Compiling

Faunus compiles on most unix-like systems including Linux, MacOS and most likely also in a Cygwin environment. While detailed instructions can be found in the README file, here is a short version:

$ cd faunus/trunk
$ cmake . [options]
$ make
$ make test

Requirements:

  • CMake
  • C/C++ compiler
    (gcc,clang,intel,xlc,suncc,pgcc etc.)
  • Openbabel (optional)
  • OpenMP for parallellization (optional)
  • Doxygen for coding manual (optional)
