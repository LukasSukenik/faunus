Problems?

Faunus is developed as a programming interface to be used in your own C++/Python code. We do however provide a number of sample programs that can be used more or less without programming. Should you encounter problems or have questions, please use our forums at Sourceforge:

  • General questions - Help with running, compiling etc.
  • Developer support - Developer discussions, bugs, design.
  • Feature requests - Any suggestions you would like to see or need help developing?
